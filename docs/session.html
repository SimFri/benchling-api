Session (benchlingapi.session)¶

Main entrypoint for the benchling api. To open a session, use your Benchling-provided API key. From there, you can access the session-enabled models:

from benchlingapi import Session

session = Session("asdfe8iunaoerhgoaher")
session.DNASequence
session.AASequence
session.Oligo
session.Registry

For more information on the models and their methods, see the API model docs
