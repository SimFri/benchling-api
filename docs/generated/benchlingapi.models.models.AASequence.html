classmethod list(page_size=None, next_token=None, sort=None, amino_acids=None, archive_reason=None, folder_id=None, modified_at=None, name=None, project_id=None, schema_id=None, registry_id=None, **kwargs)[source]¶

List AASequence models.

Parameters

  • page_size (Optional[int]) – Number of results to return. Defaults to 50, maximum of 100.

  • next_token (Optional[str]) – Token for pagination

  • sort (Optional[str]) –

    Method by which to order search results. Valid sorts are modifiedAt (modified time, most recent first) and name (entity name, alphabetical).

    Optionally add :asc or :desc to specify ascending or descending order. Default is modifiedAt.

  • archive_reason (Optional[str]) – Archive reason. Restricts DNA Sequences to those with the specified archive reason. Use “NOT_ARCHIVED” to filter for unarchived DNA Sequences.

  • folder_id (Optional[str]) – ID of a folder. Restricts results to those in the folder.

  • modified_at (Optional[str]) – Datetime, in RFC 3339 format. Supports the > and < operators. Time zone defaults to UTC. Restricts results to those modified in the specified range. e.g. > 2017-04-30.

  • name (Optional[str]) – Name of a DNA Sequence. Restricts results to those with the specified name.

  • project_id (Optional[str]) – ID of a project. Restricts results to those in the project.

  • schema_id (Optional[str]) – ID of a schema. Restricts results to those of the specified schema.

  • registry_id (Optional[str]) – ID of a registry. Restricts results to those registered in this registry.

Returns

register(naming_strategy='NEW_IDS')¶

Register this instance.

Parameters

naming_strategy –

naming strategy for registry. - NEW_IDS (default): Generates new registry IDs, and leaves entity name

as-is

  • IDS_FROM_NAMES: Converts names into registry IDs

  • DELETE_NAMES: Generates new registry IDs and sets entity name to

    new registry ID, and does not keep old names as aliases

  • SET_FROM_NAME_PARTS: Generates new registry IDs and generates

    new entity name based on the entity schema’s name template, if it has one. :return: model instance
