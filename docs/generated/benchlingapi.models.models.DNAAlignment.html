classmethod submit_alignment(algorithm, name, template, filepaths=None, sequences=None, rawfiles=None)[source]¶

Submit an sequence alignment task.

Usage:

task = session.DNAAlignment.submit_alignment(
    algorithm='mafft',
    name='my sequence alignment',
    filepaths=[
        'data/13dfg34.ab1'          # filepath to ab1 files
    ],
    sequences=[
        'seq_1erv452',              # a benchling sequence id
        session.DNASequence.one(),  # ...or a DNASequence instance
    ],
    rawfiles=None                   # only use if you have base64 data handy
)

# wait until the alignment is finished
task.wait()

# print the alignment
print(task.response)

# or grab the alignment
alignment = task.response_class

# from there, you can delete the alignment
alignment.delete()
Parameters

  • algorithm (str) – either ‘mafft’ or ‘clustalo’

  • name (str) – name of the sequence alignment

  • template (Union[str, DNASequence]) – template of the alignment. Either a benchling sequence id (str) or a DNASequence instance containing a id.

  • filepaths (Optional[List[str]]) – optional list of sequencing filepaths to align to the template. For example, a list of .ab1 file paths.

  • sequences (Optional[List[Union[str, DNASequence]]]) – optional list of DNASequence instances or benchling sequence ids to align to the template sequence.

  • rawfiles (Optional[List[str]]) – optional list of raw entries to send to align to the template. Use this if you have base64 bytes encoded data. Take a look at Benchling V2 API for more information.
