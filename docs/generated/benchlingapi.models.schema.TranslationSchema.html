_do_load(data, many=None, partial=None, unknown=None, postprocess=True)¶

Deserialize data, returning the deserialized result.

Parameters

  • data – The data to deserialize.

  • many (bool) – Whether to deserialize data as a collection. If None, the value for self.many is used.

  • partial (bool|tuple) – Whether to validate required fields. If its value is an iterable, only fields listed in that iterable will be ignored will be allowed missing. If True, all fields will be allowed missing. If None, the value for self.partial is used.

  • unknown – Whether to exclude, include, or raise an error for unknown fields in the data. Use EXCLUDE, INCLUDE or RAISE. If None, the value for self.unknown is used.

  • postprocess (bool) – Whether to run post_load methods..

Returns

load(data, many=None, partial=None, unknown=None)¶

Deserialize a data structure to an object defined by this Schema’s fields.

Parameters

  • data (dict) – The data to deserialize.

  • many (bool) – Whether to deserialize data as a collection. If None, the value for self.many is used.

  • partial (bool|tuple) – Whether to ignore missing fields. If None, the value for self.partial is used. If its value is an iterable, only missing fields listed in that iterable will be ignored. Use dot delimiters to specify nested fields.

  • unknown – Whether to exclude, include, or raise an error for unknown fields in the data. Use EXCLUDE, INCLUDE or RAISE. If None, the value for self.unknown is used.

Returns

A dict of deserialized data

Return type

dict

New in version 1.0.0.

Changed in version 3.0.0b7: This method returns the deserialized data rather than a (data, errors) duple. A ValidationError is raised if invalid data are passed.
