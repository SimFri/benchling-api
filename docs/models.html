Models (benchlingapi.models.models)¶

This module contains the Benchling API models. Models are accessed from a session instance as follows:

from benchlingapi import Session

session = Session("asdfe8iunaoerhgoaher")
session.DNASequence
session.AASequence
session.Oligo
session.Registry
# and so on
